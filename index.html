Badanie przebiegu zmienności funkcji

\[Funkcja: f(x)=\frac{x^{2}+x+1}{x^2+2x+2}\]

Dziedzina

\[D_f=\mathbb{R}\]

Miejsca zerowe

\[x^2+x+1=0\\ x\notin R\]

Punkt przecięcia z osią OY

\[f(0)=\frac{0^2+0+1}{0^2+0+2}=\frac{1}{2}\\ (0,\frac{1}{2})\]

Parzystość, nieparzystość

\[Parzystość:\\ \exists\ x=1, f(x) \neq f(-x)\\ f(1) = \frac{1^2+1+1}{1^2+2+2}=\frac{3}{5}\\ f(-1) = \frac{(-1)^2-1+1}{(-1)^2-2+2}=1\\ Funkcja\ nie\ jest\ parzysta\\\]

\[Nieparzystość:\\ \exists\ x=1, f(x) \neq -f(-x)\\ f(1) = \frac{1^2+1+1}{1^2+2+2}=\frac{3}{5}\\ -f(-1) = -\frac{(-1)^2-1+1}{(-1)^2-2+2}=-1\\ Funkcja\ nie\ jest\ nieparzysta\]

Granice

\[\lim\limits_{x\to\infty}f(x)=1\\ \lim\limits_{x\to-\infty}f(x)=1\]

Asymptoty

\[pozioma: y= 1\\ ukośna: brak\\pionowa: brak\]

Przedziały monotoniczności

\[f'(x)=\frac{x^2+2x}{(x^2+2x+2)^2}\]

\[dla \ każdego\ x\in D_f, (x^2+2x+2)^2 > 0 \Rightarrow Znak\ pochodnej\ zależy\ od\ licznika\]

\[f'(x) = 0 \\ x^2+2x =x(x+2)= 0\]

\[x=0\ \vee \ x=-2\]

\[f'(x) > 0\ dla\ x \in (-\infty, -2),\ (0,\infty)\]

\[f(x)\ rośnie\ w\ przedziale\ x \in (-\infty, -2),\ (0,\infty)\]

\[f'(x) < 0\ dla\ x \in (-2,0)\]

\[f(x)\ maleje\ w\ przedziale\ x \in (-2,0)\]

Ekstrema lokalne

\[dla\ x\ = -2 \ \ y_{max} = f(-2) = \frac{3}{2}\\ dla\ x = 0 \ \ y_{min} = f(0) \ =\frac{1}{2}\]
